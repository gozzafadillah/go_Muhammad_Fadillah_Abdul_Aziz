# Introduction Rest API 

## Apa itu API dan Rest-API ?
Api adalah sebuah kumpulan fungsi-fungsi dan Rest-API merupakan sebuah aturan untuk memanggil sebuah fungsi-fungsi dengan metode tertentu.
